# 🧮 MathMania

## 📌 Sobre o Projeto

MathMania é um jogo simples desenvolvido em Python onde o jogador deve resolver expressões matemáticas geradas aleatoriamente.

A cada resposta correta, o jogador recebe pontos. Ao final da partida, a pontuação é registrada em um arquivo de log junto com a data e horário da execução.

## 🚀 Funcionalidades

- Geração aleatória de expressões matemáticas
- Sistema de pontuação
- Múltiplas rodadas por partida
- Validação de entradas do usuário
- Registro de pontuação em arquivo `.txt`
- Armazenamento de data e hora de cada partida

## 🛠️ Tecnologias Utilizadas

- Python
- Módulo `random`
- Módulo `time`
- Módulo `os`
- Módulo `datetime`
